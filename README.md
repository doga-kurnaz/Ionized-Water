# 🩺 The Case of Iodine

A comprehensive data analysis report exploring global **Iodized Salt Consumption** and its public health implications using **Quarto** and **Python**.

## 🚀 Overview
- **Author:** Doga Kurnaz
- **Tools:** Quarto, Python (Polars, Plotly)
- **Key Focus:** Global iodine coverage, regional disparities, and economic correlations.

## 📊 Key Features
- **Interactive Visuals:** Dynamic maps and charts showing iodine usage.
- **Data-Driven Insights:** Analysis of salt iodization trends across different income levels.
- **Reproducible:** Built with Quarto for seamless integration of code and narrative.
